Examen: MediSistema

Descripción del examen
El objetivo de este examen es diseñar e implementar una base de datos que permita gestionar la
información de médicos, empleados y pacientes de un centro de salud. El sistema debe
controlar el personal médico (titulares, interinos y sustitutos), sus horarios de consulta, los
períodos de sustitución y el registro detallado de las vacaciones planificadas y disfrutadas.
Problema
El centro de salud requiere una solución centralizada para organizar la información del personal
y los pacientes. Actualmente, la falta de un sistema genera dificultades para identificar quién
está activo, quién se encuentra en periodo de vacaciones o sustitución, y la asignación de
pacientes. El sistema debe automatizar el cálculo de horas de consulta y la gestión de periodos
de descanso para eliminar errores manuales.
